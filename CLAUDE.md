# Claude Researcher Workspace

This workspace is for everyday research and life decisions.

@rules/research-workflow.md

## Quick start

```
/research <any topic>     — full structured research
/compare <A> vs <B>       — side-by-side comparison
/decide <situation>       — decision with trade-offs and recommendation
```

## Rules

- Never fabricate data or statistics. If you can't find reliable data, say so.
- Always search in the user's language/region first, then expand if needed.
- Flag information older than 2 years as potentially outdated.
- If a decision has financial, health, or legal implications, recommend a professional.
